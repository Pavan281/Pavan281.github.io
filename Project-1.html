COVID based Chatbot using Python

This project will helps the people to understand about the Covid Symptoms with the help of chatbot.

Introduction

With the help of this project, People can use this chatbot to get some information about the COVID-19 symptoms, basic needs, hospitals etc. This Project is based on Tkinter using python for GUI and AI BOT using JSON file for AI.

Additional Information

We proposed a system which is going to work as an application and gives specific and basic information about the present health conditions. This system finds out what the user actually wants by retrieving the semantic meaning of this query. Then our system will pass on this semantic text as input to the pattern matching algorithm. The patching algorithm make user of knowledge that is already fed into it in order to generate a response.
